## About

This is a simple HTML client application demonstrating tools taught in Programming 5 course.

This project includes:

- npm
- webpack
- ESLint
- dprint
- Sass
- Bootstrap

## How to run

After downloading the project, first run:

```
npm install
```

and then:

```
npm start
```

to sucessfully run the client application. Make sure the backend is running first.
